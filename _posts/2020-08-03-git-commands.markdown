Diff for not staged changes
```
git diff
```

Diff for staged changes
```
git diff --staged
```

Unstage any staged changes for the given file
```
git reset -- <filePath>
```

Back to previous branch
```
git checkout -
```

Stash changes
```
git stash
```

Pop stashed changes
```
git stash pop
```
